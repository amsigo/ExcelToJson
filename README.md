Excel to Json
========================
 
 Repository를 받고 JsonConverter Folder의 template 엑셀 파일을 수정하여 table 폴더에 추가하고 JsonConverter.exe를 실행하면 엑셀 파일이 json파일로 변환되어
 output 폴더에 json 파일로 추출됩니다.
